# Ideas Inbox

Drop raw ideas here. One markdown file per idea. When an idea is approved, it gets promoted to a project under `poc/`.

## Format

Each idea file should follow this simple format:

```
# Idea: [Title]

## What
Brief description of the idea.

## Why
Why is this worth exploring?

## Scope
Rough estimate of effort: small / medium / large

## Status
`new` | `evaluating` | `approved` | `rejected` | `promoted to poc/[project-name]`
```

## Workflow
1. Anyone on the team can drop an idea file here
2. Hermes (orchestrator) reviews and evaluates
3. Approved ideas get a folder created under `poc/` using the template
4. The idea file gets updated with `promoted to poc/[project-name]`
